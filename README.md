# 1 milion de fapte bune — Brandbook

Brandbook interactiv (RO/EN) pentru mișcarea **1 milion de fapte bune**.

## Vezi online
Deschide `index.html` sau, după publicare pe GitHub Pages:
`https://<utilizator>.github.io/<repo>/`

## Conținut
- `index.html` — brandbookul complet (21 slide-uri, self-contained, fonturi + imagini incluse)
- `logos.zip` — logo-uri (SVG + PNG transparent + PNG color)
- `icons.zip` — set de iconuri
- `illustrations.zip` — ilustrații

## Publicare pe GitHub Pages
1. Creează un repo nou (ex. `brandbook`) și urcă toate fișierele din acest folder.
2. Settings → Pages → Source: `Deploy from a branch` → branch `main` / folder `/ (root)`.
3. După ~1 minut, linkul `https://<utilizator>.github.io/<repo>/` e gata de trimis pe WhatsApp.
